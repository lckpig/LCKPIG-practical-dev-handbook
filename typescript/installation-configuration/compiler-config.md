# Configuración básica del compilador (`tsconfig.json`)

- Generación de `tsconfig.json` con `tsc --init`
- Parámetros esenciales (`target`, `module`, `strict`, `outDir`, `rootDir`)
- Compilación incremental con `incremental: true`
